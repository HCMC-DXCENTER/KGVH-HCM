# Không gian Văn hóa Hồ Chí Minh

Website giới thiệu và khai thác tư liệu về cuộc đời, sự nghiệp và tư tưởng Chủ tịch Hồ Chí Minh.

## Các trang

- `index.html` — Trang chủ
- `tu-lieu.html` — Tin tức, video, audio, ảnh tư liệu
- `thu-vien.html` — Thư viện sách điện tử
- `bao-tang-3d.html` — Bảo tàng và di tích 3D
- `ke-hoach.html` — Dòng đời theo triển lãm ảnh
- `lien-he.html` — Liên kết nguồn tham khảo

## Chạy trên máy

Mở `index.html` bằng trình duyệt, hoặc:

```bash
python3 -m http.server 8080
```

Truy cập: http://localhost:8080
